# GenAI Agent for Universal, Dimension Based Data Quality Scoring in Payments domain
The AI agents analyze transaction CSV quality and deliver trusted analytical datasets with actionable recommendations to enhance data readiness for RBI reporting, fraud detection, and credit decisions.
This system gives output in daily language model updates with fixed recommendations, processes data entirely in-memory (storing only scores and metadata), retains zero transaction data, and provides role-specific outputs for customers and bank employees.
## Tools Required
* CrewAI
* Great Expectations
* Polars
* Pydantic
